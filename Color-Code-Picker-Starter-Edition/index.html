Color Code Picker is an application built to help developers write code faster. It provides the color code to the developer, who copies the code and paste in their projects. This is an open source application, there's is also a pro version which comes with more functions

  • The RGB feature displays a list of selected RGB (Red, Green, Blue) color code whenever the user clicks on the button. In the pro edition of this application, the RGB feature randomly generates the code randomly there's no limit to the RGB color codes generated.
  • The HEX feature displays a list of selected HEX color code whenever the user clicks on the button. In the pro edition of this applicationn the HEX feature randomly generated hex color code whenever the user clicks on the button.
  • The GRADIENT-FX feature displays a list of selected linear-gradient color codes whenever the user clicks on the button.
  • The Pro-Edition of this application comes with more features, here are some list of the new features and what they do
  • The RGB-RANDOM feature displays a list of randomly generated RGB color codes whenever the user clicks on the button.
  • The Gradient-RANDOM feature displays a list of randomly generated linear-gradient color codes whenever the user clicks on the button. We aslo have the Deg- and Deg button, what this does is whenever either button is clicked on it adds a degree property from (10deg-360deg, -10deg/-360deg) to the randomly generated linear-gradient code.
  • The Gradient-RANDOM feature displays a list of randomly generated radial-gradient color codes whenever the user clicks on the button. We aslo have the Deg- and Deg button, what this does is whenever either button is clicked on it adds a degree property from (10deg-360deg, -10deg/-360deg) to the randomly generated radial-gradient code.
