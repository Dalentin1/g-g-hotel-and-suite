Color Code Picker is an application built to help developers write code faster. It provides the color code to the developer, who copies the code and paste in their projects. This is an open source application, this is the pro version of this application.

  • The RGB-RANDOM feature displays a list of randomly selected RGB (Red, Green, Blue) color code whenever the user clicks on the button.
  • The HEX feature displays a randomly generated hex color code whenever the user clicks on the button.
  • The GRADIENT-FX feature displays a list of selected linear-gradient color codes whenever the user clicks on the button.
  • The Gradient-RANDOM feature displays a list of randomly generated linear-gradient color codes whenever the user clicks on the button. We aslo have the Deg- and Deg button, what this does is whenever either button is clicked on it adds a degree property from (10deg-360deg, -10deg/-360deg) to the randomly generated linear-gradient code
